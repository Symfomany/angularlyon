### Angular Lyon built in Ionic 2 

Project  Demo on Ionic 2


+ Export Android Home

```
export ANDROID_HOME=/home/boyer/Documents/android-sdk-linux
```

+ Run a new project:

```
 ionic start test --v2
```

+ Run on serve

```
ionic serve
```

+ Run on Mobile

```
ionic run android --device
```
